## Description

Panel API that monthly displays the data of each client of the [Vigilantes do Sono](https://www.vigilantesdosono.com/) platform.

## Languages ​​and technologies

- NodeJs,
- Express,
- TypeScript
- NestJs
- Prisma
- SQL
## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
